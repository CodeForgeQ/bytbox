# 💻 BytBox OS

A retro-styled in-browser terminal playground.
Type commands, explore a virtual filesystem, edit files with a built-in editor, change themes, and pretend you're inside a cyberpunk operating system.

## 🟢 Status: **In active development – new features coming soon!**
## 🔴 Try it live on GitHub Pages: **COMING SOON**

## ✨ Features

- Full terminal emulation with CRT scanlines and customizable neon themes
- Virtual filesystem (look, go, mk, kick, yeet, slide, clone, dog)
- Built-in text editor tiny (save with Ctrl+S, exit with Ctrl+X)
- User management (adduser, setpass, become)
- Superuser command sumo with /etc/sumoers privileges
- Live theme customization (paint)
- ASCII art and fun commands (moo, specs, luck, rain)
- Sound effects for keystrokes, actions, and ambiance
- Install a fake OS (get "OS" -version X) that opens a window

## 💿 Quick start

Just open index.html in your browser (use a local server for full functionality).
